# Pile Programming Language Documentation

## Introduction
Pile is a concatenative scripting programming language, made to make stack-oriented programming useful in some way.

This project is still in development but it is definitely a usable (and [Turing-complete](https://en.wikipedia.org/wiki/Turing_completeness), see [`gol.pile`](../examples/gol.pile)) language.

The final goal of this language is to be a support scripting language. From a simple "calculator" language to (potentially) a powerful tool for Automation or Data analysis.
